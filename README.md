Опис ідеї та обґрунтування вибору активу:

Об'єктом дослідження обрано акції Alphabet Inc. GOOGL. Аналіз проводиться за останні два роки з денним таймфреймом на основі даних з фінансового API yfinance.

Alphabet Inc. є однією з найбільших технологічних компаній у світі, а її акції дуже популярні серед інвесторів. Через те що їх щодня купують і продають у величезних обсягах, різниця між ціною купівлі та продажу мінімальна, що робить ці акції зручними для програмування торгових роботів. Ціни на акції технологічних гігантів зазвичай рухаються в одному напрямку досить довго — цьому сприяють новини про розвиток штучного інтелекту, фінансові звіти компанії та ситуація в економіці. Оскільки ціна стійко зростає або падає протягом тривалого часу, наша стратегія з перетином двох ліній (ковзних середніх) чудово підходить, щоб вчасно помітити зміну тренду.

Процес аналізу та використовувані інструменти:

Для реалізації проєкту використовувався Python та бібліотеки yfinance для завантаження ринкових даних, pandas для очищення даних, розрахунку індикаторів та прибутку, а також matplotlib для візуалізації результатів.
Процес починається з завантаження даних по GOOGL за два роки. Далі розраховуються коротка ковзна середня SMA за 20 днів та довга ковзна середня LMA за 50 днів. Після цього очищаються заголовки колонок. На основі порівняння SMA та LMA генеруються торгові сигнали. Якщо коротка середня більша за довгу, формується сигнал Купуємо. Якщо коротка середня менша за довгу, формується сигнал Продаємо. За відсутності даних формується сигнал Утримуємось. Далі проводиться симуляція торгівлі, розраховується щоденна зміна ціни за дні утримання позиції та фіксуються дати й ціни входу та виходу.
